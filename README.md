This paper addresses the critical challenge of accurate and efficient brain tumor detection from Magnetic Resonance Imaging (MRI) scans, a task often hindered by time-consuming manual interpretation, inter-observer variability, noise, and low tissue contrast. We leverage the advancements in machine learning (ML) and deep learning (DL) to automate and enhance this process. This study systematically compares the performance of classical ML models, including Support Vector Machines (SVM) and Random Forests, with Convolutional Neural Network (CNN) architectures for brain tumor classification.
